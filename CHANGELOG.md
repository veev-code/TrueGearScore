# TrueGearScore Changelog

## [0.1.0] - 2026-03-20

### Added
- Core scoring engine with stat-weight-based gear scoring
- Cap-aware stat weights (hit, expertise diminish past cap)
- Per-class/spec stat weight tables (PvE)
- Gem, enchant, and proc databases (starter sets)
- Self-scanner: automatic rescoring on gear change
- Character panel (paperdoll) score display
- `/tgs` slash command with score and breakdown output
- Score color brackets calibrated to TacoTip GearScore ranges
